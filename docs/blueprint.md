# **App Name**: ResumeAI

## Core Features:

- Resume Upload: Allow users to upload their resume in PDF format.
- ATS Score Analysis: Analyze the uploaded resume to determine its Applicant Tracking System (ATS) score.
- Skill Gap Identification: Identify missing skills based on the analysis of uploaded resume and suggested job role, and suggest skills to learn or improve.
- Job Role Matching: Match the resume with relevant job roles based on skills, experience, and keywords using an AI tool.
- Personalized Feedback: Generate personalized suggestions using an AI tool, about how the candidate can improve their resume
- Results Dashboard: Presents data visualization and an interactive dashboard showing the AI's results in a single place

## Style Guidelines:

- Primary color: Deep sky blue (#3081D0) for trustworthiness and clarity.
- Background color: Light-blue gray (#E8F3FF) for a clean, professional background.
- Accent color: Slate blue (#6A5ACD) to highlight key metrics and actionable insights.
- Body and headline font: 'Inter', a sans-serif font, for readability and modern feel.
- Use simple, clear icons to represent data points and suggestions.
- Use subtle, non-intrusive animations to provide feedback and guide the user through the analysis.
